# Mending Mastery
> A Minecraft mod focused on exploration, spellcasting, and crafting.  

Made for **Fabric 1.21.10**

## Gameplay
<div style="display: flex;">
<img alt="infuser crafting recipe" src="/misc/screenshots/craft_infuser.png" style="width: 30%;">
<img alt="infusing process" src="/misc/screenshots/infuse.png" style="width: 30%;">
</div>

1. Craft an infuser from resin, sculk sensor, and a nether star.
2. Infuse a stick with a diamond to get a magic wand.
3. Use the magic wand to cast spells.
